# 🧠 Practical Python and OpenCV – Practice Repository

This repository contains my hands-on practice and implementations of learning from:

📘 *Practical Python and OpenCV* by Dr. Adrian Rosebrock

---

## 🚀 Overview

This repo includes implementations of core computer vision and image processing techniques using **Python** and **OpenCV**.

Each file represent a specific concept.

---

## 📂 Topics Covered

- Image loading, displaying, and saving  
- Pixel access and manipulation  
- Drawing shapes and working with ROI  
- Geometric transformations (translation, rotation, resizing, flipping)  
- Image arithmetic and bitwise operations  
- Masking techniques  
- Color spaces (BGR, grayscale, HSV)  
- Channel operations (split and merge)  
- Histograms (grayscale, color, masked)  
- Histogram equalization  
- Smoothing and blurring (Gaussian, median, bilateral)  
- Thresholding (simple, adaptive, Otsu)  
- Edge detection (Sobel, Laplacian, Canny)  
- Contour detection and basic object extraction
